ОБЩИЕ СВЕДЕНИЯ
——————————————————————————————————————————————————————————

IP Geo — Модуль для получение геолокации пользователя по его IP адресу.


УСТАНОВКА
——————————————————————————————————————————————————————————

  Процедура установки обычная, за исключением того, что нужно
  в ручную импортировать в базу две таблицы из директории ipgeo/base.
  
  mysql -uUSERNAME -pPASSWORD DBNAME < ipgeo_blocks.sql
  mysql -uUSERNAME -pPASSWORD DBNAME < ipgeo_regions.sql


ИСПОЛЬЗОВАНИЕ
——————————————————————————————————————————————————————————

  Модуль сам по себе ничего не делает, а только предоставляет возможность
  другим модулям и темам определять местонахождение пользователя по его IP адресу.
  
  Используйте ipgeo_get_region() для получение информации о регионе пользователя.
